лҸҷмқјн•ң м§Ҳл¬ём—җ лҢҖн•ҙм„ңлҠ” мәҗмӢұлҗң лӢөліҖмқ„ лӮҙліҙлӮҙлҠ” кІғмңјлЎң нҶ нҒ° л№„мҡ©мқ„ мӨ„мқҙлҠ” л°©лІ•мқ„ л§җн•ңлӢӨ. лҸҷмқјн•ң м§Ҳл¬ёмқҙ м–ём ң л“Өм–ҙмҳ¬м§Җ, лҳҗ лҸҷмқјн•ң м§Ҳл¬ёмқҙлқјлҠ”кұҙ нҶ м”Ён•ҳлӮҳ м•ҲнӢҖлҰ¬кі  лҸҷмқјн•ң м§Ҳл¬ёл§Ңмқ„ л§җн•ҳлҠ”кұҙм§Җ к¶ҒкёҲн•ҳкё°лҠ” н•ҳлӢӨ.

мәҗмӢұ л°©лІ•

м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң мәҗмӢұмқ„ мң„н•ҙ мӮ¬мҡ©н•  мҲҳ мһҲлҠ” л°©мӢқмңјлЎңлҠ” мқёл©”лӘЁлҰ¬ мәҗмӢңмҷҖ sqlite мәҗмӢңк°Җ мһҲлӢӨ.

  • мқёл©”лӘЁлҰ¬ мәҗмӢң : л©”лӘЁлҰ¬ кіөк°„мқ„ нҷңмҡ©н•ҙ лҸҷмқјн•ң м§Ҳл¬ём—җ лҢҖн•ң лӢөліҖмқ„ мқјмӢңм ҒмңјлЎң м ҖмһҘн•Ё. лҸҷмқјн•ң мҡ”мІӯмқҙ л°ҳліөлҗ  кІҪмҡ° мәҗмӢңлҗң мқ‘лӢөмқ„ мҰүмӢң л°ҳнҷҳ. н”„лЎңк·ёлһЁ мў…лЈҢмӢң л©”лӘЁлҰ¬к°Җ нңҳл°ңлҗЁ мӮ¬мҡ©мһҗлі„ л§һм¶Өнҳ• м§Ҳл¬ёмқҙлӮҳ лЎңк·ём•„мӣғмӢң мҙҲкё°нҷ”к°Җ н•„мҡ”н•ң кІҪмҡ°м—җ мң мҡ©. кі к°қм„јн„°, AS м„јн„° л“ұ мһҗмЈј л°ҳліөлҗҳлҠ” м •нҳ•нҷ”лҗң м§Ҳл¬ёкіј лӢөліҖм—җ м Ғн•©н•ң л°©мӢқ
  • SQLITE мәҗмӢң : лҚ°мқҙн„°лІ мқҙмҠӨ нҢҢмқјмқ„ нҷңмҡ©н•ҙ мәҗмӢң лҚ°мқҙн„°лҘј м ҖмһҘн•ҳм—¬ л©”лӘЁлҰ¬к°Җ нңҳл°ңлҗҳм§Җ м•ҠмқҢ ``` from langchain_openai import ChatOpenAI from langchain_core.prompts import PromptTemplate

llm = ChatOpenAI(model_name=вҖқвҖҰвҖқ)

prompt = PrompotTemplate.from_template(вҖң{country}м—җ лҢҖн•ҙм„ң 200мһҗ лӮҙмҷёлЎң мҡ”м•Ҫн•ҙмӨҳвҖқ)

chain = prompt llm

from langchain.globals import set_llm_cache from langchain.cache import InMemoryCache
