문제

올바르게 괄호가 닫혀있는지 판단하기

해결

def solution(s):
    stack = []
    for p in s:
        if p == '(':
            stack.append(p)
        else:
            if not stack:
                return False
            stack.pop()
    return not stack

알게된 점

  • 스택에 넣고 마지막에 넣은 값을 조건문에 넣어서 비교해서 빼는 방식으로 진행되는 것이 포인트.
