мӨ‘ліөлҗҳлҠ” мӣҗмҶҢк°Җ м—ҶлҠ” нҠңн”Ңмқҙ мһҲкі  мқҙкұё нҶ лҢҖлЎң л¶Җл¶„м§‘н•©мқ„ л§Ңл“Өм–ҙм„ң мӨ¬мқ„л•Ң, мӣҗлһҳмқҳ нҠңн”Ң м¶”лЎ н•ҳкё°.

н’Җмқҙ

def solution(s):
    # нҢҢмӢұ
    ## л¬ёмһҗм—ҙм—җм„ң м§‘н•©нҳ•нғң м ңкұ°н•ҳкі  л¶„лҰ¬н•ҳкё°
    s = s[2:-2].replace('},{','-')
    s = s.split('-')

    ## мҲ«мһҗнҳ•мңјлЎң ліҖнҷҳн•ҳкі  м…ӢмңјлЎң л§Ңл“Өкё°
    new_s = [set(map(int, x.split(','))) for x in s]

    ## кёёмқҙм—җ л”°лқј мҳӨлҰ„м°ЁмҲң м •л ¬н•ҳкё°
    new_s.sort(key=len)

    # мғҲлЎңмқҙ м¶”к°Җлҗң мӣҗмҶҢ м°ҫм•„м„ң л°ҳнҷҳн•ҳкё°
    result = []
    for subset in new_s:
        number = (list(subset-set(result))[0])
        result.append(number)

    return result
