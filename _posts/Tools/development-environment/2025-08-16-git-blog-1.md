# jekyll로 깃 블로그 만들기

작년에 처음 만든 내 깃 블로그.. 처음에 클론을 받아서 이것저것 고쳐가면서 만들었지만 아쉬운 부분이 많았고, 자바 스크립트에 대해서 잘 몰랐기 때문에(지금도 잘 모름) 포스팅만 좀 열심히 하다가 내버려둔 측면이 있었다..

지금은 스터디도 하고 있고, 공부하는 내용을 기록해야한다는 생각이 들어서 블로그부터 고쳐서 써보기로 했다!
그래서 차근차근 공부해서 바꿔가는 깃 블로그 일기.

## jekyll이란?

![about jekyll -- git docs](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/about-github-pages-and-jekyll)

깃헙 페이지스를 기본적으로 지원하는 정적 사이트 생성기. 마크다운과 html 파일을 사용하여 사용자가 선택한 레이아웃을 기반으로 정적 웹사이트를 생성한다.

## 프로젝트 구조

> \_config.yml
> 사이트 테마 및 플러그인 등 대부분의 지킬 설정은 config 파일을 편집하여 구성한다.
> 기타 다양한 설정에 대해서는 이 [문서](https://jekyllrb.com/docs/configuration/)를 참고하면 좋을것 같다.
