1л¶Җм—җм„ң Jekyllкіј LiquidлЎң м№ҙн…Ңкі лҰ¬ мӢңмҠӨн…ңмқҳ HTML кө¬мЎ°лҘј л§Ңл“Өм—ҲмҠөлӢҲлӢӨ. мқҙлІҲм—җлҠ” JavaScriptлҘј мӮ¬мҡ©н•ҙм„ң мӮ¬мҡ©мһҗк°Җ мӢӨм ңлЎң нҒҙлҰӯн•ҳкі  мғҒнҳёмһ‘мҡ©н•  мҲҳ мһҲлҠ” лҸҷм Ғ кё°лҠҘмқ„ кө¬нҳ„н•ҙліҙкІ мҠөлӢҲлӢӨ.

DOMкіј мқҙлІӨнҠё лҰ¬мҠӨл„Ҳ кё°мҙҲ

лЁјм Җ DOM(Document Object Model)м—җ лҢҖн•ҙ к°„лӢЁнһҲ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ. DOMмқҖ мӣ№нҺҳмқҙм§Җмқҳ HTML кө¬мЎ°лҘј JavaScriptм—җм„ң мЎ°мһ‘н•  мҲҳ мһҲкІҢ н•ҙмЈјлҠ” мқён„°нҺҳмқҙмҠӨмһ…лӢҲлӢӨ.

DOM м„ нғқмһҗ нҷңмҡ©н•ҳкё°

// HTMLмқҙ мқҙл ҮкІҢ мһҲлӢӨл©ҙ
// <div class="subcategory-card" data-subcategory="genai-llm">
//     <h3 class="subcat-title">GenAI&LLM</h3>
// </div>

// JavaScriptм—җм„ң мқҙл ҮкІҢ м ‘к·јн•  мҲҳ мһҲм–ҙмҡ”
const card = document.querySelector(".subcategory-card");
console.log(card.dataset.subcategory); // "genai-llm"
console.log(card.querySelector(".subcat-title").textContent); // "GenAI&LLM"
